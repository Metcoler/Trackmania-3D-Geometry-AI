\chapter*{Appendix A: Contents of the Electronic Appendix}
\addcontentsline{toc}{chapter}{Appendix A}

The electronic appendix contains the basic project structure used in the thesis: the main source files, maps, 3D block models, the game plugin, an auxiliary map extractor, and selected video recordings of agent runs. The complete project repository with broader documentation is available at \url{https://github.com/Metcoler/Trackmania-3D-Geometry-AI}.

The appendix includes videos of selected agent runs on the flat track, the track with height changes, the track with different surfaces, and the comparison map \texttt{small\_map}.

The videos serve as a qualitative complement to the numerical results. They make it possible to inspect the smoothness of driving, behavior in corners, slowing down before difficult sections, and the difference between the final agent and the older bachelor-thesis version.

When interpreting the videos, it should be kept in mind that recording can slightly load the system and change the smoothness of the game. A practically deterministic evaluation can therefore behave as stochastic, where the source of variability is mainly the fluctuation of FPS and the game update rhythm. The video recordings therefore serve mainly as illustrative examples of agent behavior; the times and metrics reported in the thesis are based on logged experiments.
