# Stanford Bunny Asset

Source: Stanford 3D Scanning Repository, Stanford University Computer Graphics Laboratory.

URL: https://graphics.stanford.edu/data/3Dscanrep/

The repository states that its models may be used for research purposes and that images made using the models may be published in a scholarly article or book when the Stanford Computer Graphics Laboratory is credited. This thesis figure uses the Stanford Bunny geometry only as an illustrative mesh for explaining triangle meshes and raycasting.
